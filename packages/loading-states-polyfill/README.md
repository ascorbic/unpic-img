# @unpic/loading-states-polyfill

Browser polyfill for loading states functionality.

## Installation

```bash
npm install @unpic/loading-states-polyfill
```

## Usage

```javascript
import { init } from '@unpic/loading-states-polyfill';

// Initialize the polyfill
init();
```

## Development

```bash
# Install dependencies
pnpm install

# Build the package
pnpm build

# Run tests
pnpm test
```
