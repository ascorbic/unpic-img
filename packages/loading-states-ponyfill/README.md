# @unpic/loading-states-ponyfill

Browser ponyfill for loading states functionality.

## Installation

```bash
npm install @unpic/loading-states-ponyfill
```

## Usage

```javascript
import { init } from '@unpic/loading-states-ponyfill';

// Initialize the ponyfill
init();
```

## Development

```bash
# Install dependencies
pnpm install

# Build the package
pnpm build

# Run tests
pnpm test
```
