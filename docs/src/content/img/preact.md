## Installation and usage

```bash
npm install @unpic/preact
```

```tsx
  return (
    <Image
      src="https://cdn.shopify.com/static/sample-images/bath_grande_crop_center.jpeg"
      layout="constrained"

      height={600}
      alt="A lovely bath"
    />
  );
}
```
